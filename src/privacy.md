# Privacy

Questo sito è un sito statico informativo del condominio Via Calisina 9.

Il sito non usa analytics, non usa cookie di profilazione, non contiene moduli di contatto e non raccoglie dati personali tramite database o backend applicativi.

I contenuti sono pubblicati come pagine statiche e ospitati tramite GitHub Pages. GitHub può trattare dati tecnici necessari all'erogazione del servizio, come indicato nella documentazione e nelle informative privacy di GitHub.

Per mantenere il sito semplice e rispettoso della privacy, non devono essere aggiunti strumenti di tracciamento, risorse esterne invasive o documenti con dati personali reali senza una valutazione preventiva.
